# TEMPO Portfolio

Responsive personal portfolio built with React, Vite, Tailwind CSS, Framer Motion, GSAP, and TypeScript.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` checks the codebase with ESLint.

## Project Structure

- `src/components/portfolio` contains the page sections and motion components.
- `src/data/portfolio.ts` contains portfolio copy, project data, and navigation links.
- `src/App.tsx` composes the portfolio page.
